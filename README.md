# MT_chart_data_merge

Merge the objects in main chart of MetaTrader.

Python version 3.11

## How to use

1. Open `main.py`
2. Set the fllowing variables 

   * encoding
   * output_file_name
   * input_file_name1
   * input_file_name2

3. Run `python main.py`
